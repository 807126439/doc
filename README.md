### 常用设计模式

​	用好设计模式能帮助我们更好的解决实际问题，设计模式最重要的是解耦。设计模式天天都在用，但自己却无感知。把设计模式作为一个专题，主要是学习设计模式是如何总结经验的，把经验为自己所用。同时也为阅读框架源码打下坚实的基础。在学习设计模式之前，一定要先了解软件设计原则。
